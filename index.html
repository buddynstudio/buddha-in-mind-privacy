「마음에 새기는 부처님 말씀」(이하 "본 앱")은 사용자의 개인정보를 매우 소중히 생각합니다.
본 앱은 별도의 회원가입을 요구하지 않으며, 사용자의 개인정보를 개발자(BuddyN)의 서버로 수집하거나 전송하지 않습니다. 즐겨찾기·설정값·구매 내역 등 모든 사용자 데이터는 사용자의 스마트폰 기기 내부에만 저장됩니다.
본 앱은 광고를 일체 표시하지 않으며, 인앱결제(Google Play Billing) 기능을 위해 일부 데이터가 Google에 의해 처리될 수 있습니다. 자세한 내용은 아래 4번 항목에서 안내합니다.

1. 수집하는 개인정보 항목

본 앱은 사용자를 식별할 수 있는 개인정보(이름·전화번호·이메일·정확한 위치 정보 등)를 일체 수집하지 않습니다. 앱 내에서 생성된 즐겨찾기 목록과 설정값(알림 시간·글씨 크기·테마·듣기 속도·잠금해제된 수묵화 팩 등)은 사용자의 스마트폰 기기 내부 저장소(SharedPreferences)에만 저장되며 외부로 전송되지 않습니다.

인앱결제 기능 이용 시, 구매 영수증(purchase token)이 Google Play Billing SDK에 의해 처리됩니다. 자세한 내용은 4번 항목을 참고해 주십시오.

2. 기기 권한 안내

본 앱은 매일 정해진 시간에 부처님 말씀 알림을 전달하기 위해 아래의 기기 권한을 요청합니다.

  • 알림 표시 권한 (POST_NOTIFICATIONS): 매일 한 구절 알림을 보내기 위해 사용됩니다. 사용자가 거부할 수 있으며 거부 시 알림 기능만 사용되지 않습니다.
  • 정확한 알람 권한 (SCHEDULE_EXACT_ALARM): 사용자가 설정한 정각에 알림이 도착하도록 시스템 알람 기능을 사용합니다. Doze/Standby 모드에서도 정시 알림을 위해 필요합니다.
  • 부팅 완료 수신 (RECEIVE_BOOT_COMPLETED): 스마트폰 재부팅 후에도 알림이 자동으로 복구되도록 사용됩니다.
  • 화면 잠금 해제 유지 (WAKE_LOCK): 알림 전달 시점에 시스템이 짧게 깨어나기 위해 사용됩니다.
  • 진동 (VIBRATE): 알림 도착 시 사용자가 인지하도록 짧게 진동하기 위해 사용됩니다.
  • 인터넷 접속 (INTERNET): 인앱결제(Google Play Billing) 기능을 위해 사용됩니다. 핵심 콘텐츠(부처님 말씀 본문·이미지·즐겨찾기·알림·TTS)는 인터넷 없이도 작동합니다.

3. 앱 내 기능 및 데이터 처리 방식

본 앱의 각 기능은 다음과 같이 데이터를 처리합니다.

  • 부처님 말씀 표시: 앱 내에 사전 탑재된 정적 콘텐츠(법구경·금강경 등 공개 본문 127편)를 보여줍니다. 외부 서버 호출이 없습니다.
  • 즐겨찾기 / 설정값 / 잠금해제 정보 저장: 사용자가 즐겨찾기 한 말씀, 설정값(알림 시간·글씨 크기·테마·듣기 속도), 잠금해제된 수묵화 팩, TTS Pro 활성 상태 등은 모두 스마트폰 기기 내부 저장소(SharedPreferences)에만 저장됩니다. 앱을 삭제하면 해당 데이터도 영구적으로 함께 삭제됩니다.
  • 듣기 (TTS):
    • 무료 사용자: 스마트폰에 내장된 시스템 TTS 엔진(flutter_tts)을 통해 처리됩니다. 본 앱이 직접 외부로 텍스트를 전송하지 않으나, 스마트폰 제조사 및 시스템 TTS 엔진의 자체 정책은 별도로 적용됩니다.
    • TTS Pro 평생 해금 사용자 (₩2,900 인앱결제): 앱 패키지에 사전 탑재된 mp3 음성 파일(127편, 사전 Google Cloud TTS로 생성)을 재생합니다. 재생 시점에 외부 서버 통신은 발생하지 않습니다.
  • 수묵화 컬렉션: 앱 내에 사전 탑재된 수묵화 이미지(59장)를 표시합니다. 무료 38장은 즉시 사용 가능하며, 꽃 컬렉션 4팩(연꽃·매화·벚꽃·모란, 21장)은 인앱결제로 잠금해제됩니다.
  • 공유: 사용자가 [공유] 버튼을 눌러 카카오톡 등 외부 앱으로 보낼 때, 시스템 공유 시트가 텍스트(또는 사용자가 선택한 배경 이미지와 텍스트)를 외부 앱에 전달합니다. 본 앱은 사용자가 누구에게 무엇을 공유했는지 추적하거나 기록하지 않습니다.
  • 일일 알림: 사용자가 설정한 시간에 안드로이드 시스템 로컬 알림으로 한 구절을 표시합니다. 모든 알림 처리는 기기 내부에서 이루어지며 외부 푸시 서버를 사용하지 않습니다.

  • 처리 목적: 구매 확인·환불 처리·구매 이력 복원(앱 재설치 시 자동 복원)
